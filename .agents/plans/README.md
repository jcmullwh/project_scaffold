# Plans

This repo tracks execution plans in `.agents/plans/` and todo/spec documents in `.agents/todos/`.

Directory structure:

- Plan drafts/ideas live in `.agents/ideas/`.
- Ready-to-execute plans live in `.agents/plans/2 - ready/`.
- In-progress plans live in `.agents/plans/3 - in_progress/`.
- Plans awaiting review live in `.agents/plans/4 - for_review/`.
- Completed plans live in `.agents/plans/5 - complete/`.

The current in-progress plan(s) should always live in `.agents/plans/3 - in_progress/`.
